This is the documentation of the news Black Boards API system.

For simple use, user should simply create an instance of class BlackBoardsAPI. The instance provides acccess to the three main BlackBoards through the corresponding declared objects, namely articles,feeds and outlets. Each BlackBoard member provides basic search and annotation functionality for the corresponding documents.

For example to add a tag to an article:

BlackBoardsAPI _api = new BlackBoardsAPI("noam.settings");

List<Object> arts = _api.articles.getArticles(null,null,null,null,null);
